# AI-Based Recommendation System

## Overview
This project is a machine learning-based recommendation system developed using Python and Scikit-learn. It recommends movies based on similarity between genres using content-based filtering.

## Features
- Content-based recommendation
- Cosine similarity algorithm
- Personalized suggestions
- Consumer behavior analysis

## Technologies Used
- Python
- Pandas
- Scikit-learn

## How It Works
The system analyzes movie genres and recommends similar movies based on user preferences.
